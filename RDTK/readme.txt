how to use RDTK

1. to plot your graph, click in the button "abrir arquivo csv", it will run the def "abrirarq" in the code, 
2. it will generate a frame with checkbuttons that are the name of your excel columns first row
3. select the channels clicking in the checkbutton, and then click the button "plotar grafico" to plot the graph
4. if you want to add a math channel to analyse, click the button "canal matematico", it will run the def "canal_mat" in the code
5. it will open a new window (maths), in this window you can select 2 channels and one of the 4 basics operators to do your math channel
6. name your math channel writing in the "nome do canal matematico:" text box
7. thats it, hope you enjoy it :)
